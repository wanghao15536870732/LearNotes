# Java简介

## 第一个Java程序：永远的“Hello World !”

### 定义一个新的文件：Hello.java

```java
public class Hello { // 定义一个类
   public static void main(String args[]) { // 主方法，一切程序的起点
        System.out.println("Hello World !") ; // 在屏幕上打印输出
    }
}
```
